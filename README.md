# 🌌 Star Wars Characters App

Aplicação web simples desenvolvida com HTML, CSS e JavaScript para buscar personagens de Star Wars, adicionar aos favoritos (like) e remover da lista.

## 🚀 Tecnologias utilizadas
- HTML
- CSS
- JavaScript 

## 📸 Funcionalidades
- 🔍 Buscar personagem pelo nome
- ⭐ Dar like em personagens
- 🗑️ Remover personagem da lista

## 🧠 Objetivo do projeto
Projeto criado com foco em prática de:
- Manipulação de DOM
- Lógica com JavaScript
- Simulação de cenários reais para testes QA
